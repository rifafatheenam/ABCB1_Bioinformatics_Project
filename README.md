# In-silico Analysis of ABCB1 Protein

This repository contains the bioinformatics analysis of the human ABCB1 protein involved in drug resistance in oral cancer.

## Objectives
- Retrieve ABCB1 protein sequence from UniProt
- Confirm protein identity using BLASTP
- Analyze physicochemical properties
- Perform structural and functional analysis

## Tools Used
- UniProt
- NCBI BLAST
- ProtParam
- Structure visualization tools
